- `HellfireRamparts_Entrance.tga`

- etc.

## How to Create Textures

### Option 1: Screenshots
1. Enter each dungeon in-game
2. Open map (`M` key)
3. Take screenshot (`Print Screen`)
4. Convert to `.tga` format using GIMP or similar
5. Resize to 512x512 or 1024x1024 (power of 2)

### Option 2: Extract from Atlas
If you have Atlas addon with TBC maps, you can reference their textures directly

### Option 3: Use In-Game Maps
The code currently shows placeholder boxes if textures are missing.
You can add textures later without breaking functionality.
